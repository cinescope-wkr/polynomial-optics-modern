## Quickstart (C++ build)

```bash
make
./bin/ex0-basicarithmetic
```

Minimal postprocess run (fast sanity settings):

```bash
./bin/ex1-postprocess path/to/input.pfm -p 1 -s 1 -o out.exr
```

## Docs (local preview)

From repo root:

```bash
python3 -m venv .venv
. .venv/bin/activate
pip install -r docs/requirements-docs.txt
mkdocs serve -f docs/mkdocs.yml
```
